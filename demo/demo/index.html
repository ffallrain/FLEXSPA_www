Flex-Site DB Description

Flex-Site DB derived from Protein Data Bank(PDB) is a specialized database for exploring the protein binding site conformational change. This database provides multiple crystal conformations for the biologically relevant binding site and the different levels of conformational changes among these structures. In order to investigate the nature of protein flexibility, several physical, chemical and structural factors, such as residue solvent accessibility, B-factors and descriptors for protein-ligand interactions were calculated and provided in Flex-Site DB.

Flex-Site DB is online freely accessible to provide the community the research tool for systematically investigating the protein flexibility using computational methods. We hope this database and analysis information provided by this study will be useful for computer-aid drug design.space
