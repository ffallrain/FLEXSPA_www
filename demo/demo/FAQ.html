FAQ:

1. What Can Flex-Site DB provide ?

  1. The summary of the protein PDB file content
  2. Multiple conformations for each single protein and their representative conformations
  3. A ready-to-go conformational ensemble for flexible molecular docking
  4. Number of binding sites in each protein, binding site location and secondary structure assignment
  5. Descriptors of protein-ligand interactions, such as conventional hydrogen bond, weak hydrogen bond, halogen bond, π-π, Rotatable Bond Score (RBScore), AHPDI (Atom hydrophobicity difference), BURCP (Ligand Buried carbon percentage), Hydrophobic contact.
  6. Binding site and secondary structure conformational changes
  7. Possible flexible residues of the binding site which is indicated by location on what type of secondary structure
  8. Possible flexible residues of the binding site which is indicated by conformational change
  9. Possible flexible residues of the binding site which is indicated by following descriptors: residue normalized B-factor, residue solvent accessibility
  10. Possible flexible residues of the binding site which is indicated by steric clash with ligand
  11. All the proteins with which the query ligand co-crystalized
  12. Several useful datasets for evaluating conformational sampling techniques
  13. Dataset containing computationally generated binding site conformations
  14. Useful Links to other database, providing other valuable information

2. Can I get the entire database ?

Yes, All of these protein and ligand structures can be downloaded. The entire database and some derived datasets can also be downloaded. We will derive more useful datasets in the future.
